CSS selectors and specificity

Selectors

Selector Example Description
.class .intro Selects all elements with class="intro"
#id #firstname Selects the element with id="firstname"
element p Selects all paraghraph (p) elements
* * Selects all elements
[attribute] [target] Selects all elements with a target attribute

Specificity Hierarchy

Categories Description
Inline styles An inline style is attached directly to the element to be styled.
IDs An ID is a unique identifier for the page elements.
Classes This category includes .classes, [attributes] and pseudo-classes.
Elements This category includes element names and pseudo-elements.
